IIDX was one of the last things keeping me dual-booting Microslop® Windows.
This project automates the entire setup (Proton-GE patching, bmsound_wine, spicetools, Steam symlinks, and .desktop launcher entries), so the game just works on Linux.

The script acts as an interactive TUI wizard with optional CLI flags.
It also supports Asphyxia server configuration for online play.
