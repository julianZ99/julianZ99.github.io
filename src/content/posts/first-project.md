IIDX was one of the last things keeping me dual-booting Windows, so I made this script that automates the entire setup (Proton-GE patching, bmsound_wine, spicetools, Steam symlinks, and .desktop launcher entries). Now the game "just works"™ on Linux.

It acts as an interactive TUI wizard with optional CLI flags and has an Asphyxia server configuration section.
