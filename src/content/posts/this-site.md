This is my personal site and blog, the one you are looking at right now. It is
statically generated with [Astro](https://astro.build/) and deployed to GitHub Pages on every push. The repo is linked above if you want to dig in.

## How posts work

- Every entry is a **post**, and the layout adapts to whatever it has. A post can be a project, a note, or just something I felt like writing.
- Add a repo and it shows the link plus the last-commit date; skip it and it is a plain blog post.
- Optional hero image and list icon, set explicitly per post.

## Details I cared about

- **Cool styling**: a terminal theme, pixel fonts, custom animations
- **Size / bloat**: static output, no heavy frameworks, ships almost no JavaScript.
- **Ease of use**: a new post is just a Markdown file with a few frontmatter fields.
